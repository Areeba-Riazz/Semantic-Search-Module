# Experiments

This directory is for **retrieval evaluation and analysis**.

Use the GUI's built-in **Evaluation & Query History** panel (Section 4) to:

- Run multiple queries with different parameters
- Compare embedding models (e.g., `all-MiniLM-L6-v2` vs `all-mpnet-base-v2`)
- Compare vector stores (FAISS vs Chroma)
- Track search latency and result counts

Export or screenshot the query log table for your report.
